# Changelog

All notable changes to the Reverse API Engineer plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-02

### Added
- Initial release of Reverse API Engineer plugin
- Manual mode command for browser-based HAR capture
- Engineer mode command for re-analyzing existing HAR files
- Agent mode command for autonomous browser navigation
- Proactive agent for automatic API reverse engineering suggestions
- Comprehensive reverse-engineering-api skill with:
  - HAR analysis guidance
  - Authentication pattern detection
  - API client generation templates
  - Testing and refinement protocols
- Integration with rae-playwright-mcp for browser control
- UUID generation and run history tracking
- Support for both run_id and direct HAR path references
